# Lily Town

## Basic Description

Lily Town is an RPG-style game that teaches the player how to administer Mental Health First Aid to a variety of characters undergoing psychological distress. This game aims to help the player learn how to efectively use Mental Health First Aid in a variety of situtations.

## Hackathon Project

This project was created by Steven Le for the MediHacks 2024 Hackathon. This project utilizes the Laerdal track with an emphasis on Scenario Simulation with Peer Feedback.
